## Setup

- Development setup:

    - Create `.env` file in root dir:

        ```text
            DEBUG=<True>
            RELOAD=<True>
            API_HOST=<0.0.0.0>
            API_PORT=<80>
            DB_USER=<user>
            DB_PASSWORD=<pass>
        ```

    - run `docker-compose up`
